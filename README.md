# Bluefy Transcription Test

iPhone の Bluefy で文字起こし方法を検証するためのテストページです。

確認できること:

- マイク入力が使えるか
- 標準の `SpeechRecognition` / `webkitSpeechRecognition` が使えるか
- 録音した音声を無料のブラウザ内 Whisper tiny で解析できるか

公開URL:

https://mukumuku-bot.github.io/Bluefy_transcription_test/

## 使い方

1. Bluefyで公開URLを開く
2. 「機能を確認」を押す
3. 「3秒録音」を押して声を録音する
4. 「標準文字起こし開始」を試す
5. うまくいかない場合は「録音した音声をAI解析」を試す

無料AI文字起こしは端末内でモデルを動かすため、iPhoneでは重い可能性があります。
